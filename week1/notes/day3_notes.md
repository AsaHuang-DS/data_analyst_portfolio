# Day 3 Notes - Intermediate SQL aggregations  + challenge query

Todays concepts:

1. Aggregations
- `SUM()`, `AVG()`, `MIN()`, `MAX()`, `COUNT()`

2. GROUP BY
Used to roll up data by customer, product, invoice, country, etc.

3. HAVING
Filter aggregated results
*(WHERE filters rows BEFORE aggregation, HAVING filters AFTER aggregation)*

4. Derived metrics
- Revenue = Quantity * Price
- Order count
- Customer frequency

5. NULL handling
- `COALESCE()`
- Null-safe comparisons

These are core analyst skills used in every industry.
